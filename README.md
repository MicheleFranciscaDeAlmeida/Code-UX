## 📌 Sobre o projeto
O **Code UX** é um projeto focado na criação de interfaces digitais intuitivas e agradáveis, aplicando conceitos de UX (User Experience) e UI (User Interface).

## 🎯 Problema que resolve
Muitas aplicações possuem interfaces confusas, pouco intuitivas e difíceis de usar, prejudicando a experiência do usuário.  
Este projeto busca resolver esse problema por meio da construção de layouts organizados, claros e centrados no usuário.

## 🎯 Objetivo
Desenvolver habilidades em design de interfaces e usabilidade, criando experiências digitais mais eficientes e acessíveis.

## 🧠 Conceitos aplicados
- Experiência do usuário (UX)
- Interface do usuário (UI)
- Hierarquia visual
- Usabilidade
- Design responsivo

## 🛠 Tecnologias utilizadas
- HTML
- CSS

## 💡 Diferenciais
- Foco na experiência do usuário
- Interface limpa e organizada
- Estrutura visual pensada para facilitar a navegação
- Aplicação de boas práticas de design

## 🌱 Conexão com sustentabilidade
O design de interfaces também impacta diretamente o consumo e o comportamento dos usuários. Interfaces mal projetadas podem gerar confusão, decisões inadequadas e até desperdício de recursos.

Este projeto aplica conceitos de UX/UI para criar experiências mais claras e intuitivas, contribuindo para que sistemas digitais incentivem decisões mais conscientes e eficientes.

A melhoria da experiência do usuário pode apoiar soluções tecnológicas voltadas à sustentabilidade, reduzindo erros, otimizando processos e promovendo o uso responsável de recursos.

## 📷 Demonstração
Este projeto é focado na estrutura e conceitos de UX/UI, com ênfase na organização visual e usabilidade da interface.

## 📚 Aprendizados
Durante o desenvolvimento, foram trabalhados:
- Organização visual de interfaces
- Criação de layouts responsivos
- Pensamento centrado no usuário
- Boas práticas de UX/UI
